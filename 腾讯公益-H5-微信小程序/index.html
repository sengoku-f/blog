个人开发的 API

一个从文字生成图片与印章的 API 接口

因为重返秦俑本色中有用户输入名字然后生成印章海报的功能需求，所以开发了个 API。

主要使用 Python 进行实现的，使用 Python 的 Python Imaging Library 库进行文字图片处理，用 flask 搭建 Web 应用框架，最后部署到阿里云服务器进行使用。期间还需要申请域名，申请域名 ssl 证书进行 https 加密

API 项目主页：生成图片与印章 API 接口

请求示例

只需要修改 type 的键值；如：测试文字，就能生成图片并上传到阿里云 oss 并返回图片链接与base64 图片给程序进行后续使用
