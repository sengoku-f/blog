🍻 感谢关注我的网站。

我是一名设计师，除了本身专业所涉及的技能 After Effects / Photoshop / Illustrator / sketch 之外，还热爱研究各种新的技术与工具，目前涉足 HTML / shell / windows batch / python / powershell 等技术栈。

我认为作为设计师不应该只掌握视觉方面的工具，或者被局限于平面或动画的范围内，更多的去了解关于计算机编程方面的知识能为你的工作带来极大的优势与便利，只有你使用过、掌握过，才能在下次需要时再次召唤它。

我时常为提供自身的工作效率而写一些自用的小工具，这些工具我都开源在 Github 上面，我的 Windows 工具合集，这些自动化的工具将我从繁复的工作中解放出来，极大的提高了我的工作效率。

本博客的文章都是我做过的作品与项目，也可以点击链接查看我的作品集，如果你喜欢，或者想与我商务上的合作，可以通过邮件📮 i.donxj@gmail.com 或者点击上方头像下的微信图标、微信号 xchacha20 联系我。

以后我还会放出更多作品，再次感谢你的关注与阅读。
